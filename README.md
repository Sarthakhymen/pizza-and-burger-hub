# Pizza and Burger Hub - Project Overview

A premium, fully responsive, multi-page restaurant website built with modern design principles, vibrant aesthetics, and seamless interactivity.

## 🚀 Key Features

*   **Responsive Multi-Page Layout**: Optimized for Desktop, Tablet, and Mobile.
*   **Stitch Design System**: Leveraged Google Stitch for a "Gourmet Editorial" UI/UX, featuring harmonious colors (#C0392B Red, #F39C12 Golden) and premium typography (Poppins).
*   **Interactive Menu**: Category filtering (Pizzas, Burgers, etc.), quantity selectors, and dietary badges.
*   **Real-time Cart**: Managed via `localStorage`, featuring a sticky sidebar on desktop and a persistent badge across all pages.
*   **Booking System**: A custom table reservation flow with time-slot availability indicators and a success confirmation modal.
*   **Order Tracking**: Visual "My Orders" page with progress steppers for active deliveries.

## 🛠️ How to View

1.  Open index.html in your browser.
2.  Navigate through the pages using the sticky navbar.
3.  Add items to the cart in the **Menu** page and try the **Checkout** flow.
4.  Reserve a table in the **Book a Slot** page to see the confirmation modal.

## 🎨 Design Philosophy

Inspired by high-end food magazines, the site uses **Tonal Layering** instead of heavy borders, **Glassmorphism** for navigation, and **Smooth Animations** (Hover lift, Scale transitions) to create a premium digital storefront.
